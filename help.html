Installation:

1) Place the 'c391Project' folder into the web_docs of your php web server.
2) You will now need to add our tables to run the site properly and handle the data. We used oracle database, and to access the database directly SQLPlus utility.
- From the command line navigate to the location where setup.sql and data.sql is located.
- Enter sqlplus and sign.
- Enter @setup.sql and the primary tables will be made.
- Enter @data.sql and the second set of tables are made and an admin account.
- Sing out of sqlplus
You now have the tables needed for the site to run
